Dash del Banco de la República es una aplicación interactiva desarrollada en Python con la librería Plotly Dash, cuyo propósito es visualizar y analizar variables económicas del Banco de la República de Colombia de manera accesible y reproducible. El proyecto permite explorar datos financieros mediante gráficos dinámicos y dashboards intuitivos, facilitando la comprensión de tendencias y resultados estadísticos. Para utilizarlo, el usuario debe clonar el repositorio desde GitHub, crear un entorno virtual con python -m venv venv y activarlo (venv\Scripts\activate en Windows o source venv/bin/activate en Mac/Linux), instalar las dependencias con pip install -r requirements.txt y finalmente ejecutar python app.py para abrir la aplicación en su navegador en la dirección http://127.0.0.1:8050
